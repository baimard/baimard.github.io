6. Installation de la solution¶

6.1. Standalone¶

6.1.1. Pré-requis¶

  1. Docker : installeur ici docker version pour vérifier

  2. docker-compose : sudo apt install docker-compose ou pip3 install docker-compose ou compose

  3. sudo apt install git

6.1.2. Installation¶

  1. copie depuis votre repository git : git clone --branch env_preparation https://git.adacis.net/b.aimard/yubico.git

  2. Créer votre fichier .env
    cd yubico
#et mettre dans un fichier .env :
USERDB=otp
PASSWORDDB=otp
DATABASEIP=database
DATABASE=otp

  3. make new

  4. make start

6.2. Haute Disponibilité¶

  1. sur un premier serveur faire l’installation Standalone

  2. sur un autre serveur copie depuis votre repository git : git clone --branch env_preparation https://git.adacis.net/b.aimard/yubico.git

  3. copier le fichier .env du premier serveur vers le serveur répliqué

  4. copier le fichier keys.json dans depuis le répertoire init_ksm du serveur de référence dans le répertoire du serveur répliqué

  5. faire la procédure de mise en HA de la base de données Base de données

  6. make start
